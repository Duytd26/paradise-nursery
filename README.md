# Paradise Nursery Shopping Application

Welcome to the Paradise Nursery Shopping Application! This is a dynamic, single-page React e-commerce application built for an online houseplant shop. 

## Features
* **Landing Page:** A welcoming entrance with company details and a seamless transition to the shopping area.
* **Product Listing:** Browse through a variety of houseplants grouped by categories (Air Purifying, Aromatic, Succulents).
* **Shopping Cart Management:** Add plants to your cart, update quantities, and remove items dynamically.
* **Redux State Management:** Efficient state handling for cart items across different components using Redux Toolkit.
* **Dynamic Pricing:** Automatically calculates total costs per item and the grand total for the entire cart.

## Technologies Used
* React.js
* Redux Toolkit
* HTML5 / CSS3
